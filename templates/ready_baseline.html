You have finished the practice without assistance, and the study will now start the main block where the game runs without the Attention Management System. In this block you decide which plate to control and when to switch between plates, and you use the gamepad joysticks and gamepad buttons as described earlier.

For each number of plates from 2 up to 9 we aim to record two valid game rounds, therefore you will play several rounds at each number of plates. After each “Game Over” screen appears you should wait for the experimenter, who will decide whether to keep that round as a valid episode or to repeat it.

When you are ready to begin the main block without assistance, please press the Start button on this page, and then use BACKSPACE on the keyboard or the left trigger (L2) on the gamepad to start each game round when the game window is open.
