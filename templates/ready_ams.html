You have finished the practice with AMS assistance, and the study will now start the main block where the Attention Management System helps you by deciding when to switch between plates. In this block you still control the active plate with the gamepad joysticks, but the AMS selects switching moments and shows you prompts that indicate the next plate.

For each number of plates from 2 up to 9 we again aim to record two valid game rounds, therefore you will play several rounds at each number of plates in the AMS-assisted condition. After each “Game Over” screen appears you should wait for the experimenter, who will decide whether to keep that round as a valid episode or to repeat it.

When you are ready to begin the main block with AMS assistance, please press the Start button on this page, and then use BACKSPACE on the keyboard or the left trigger (L2) on the gamepad to start each game round when the AMS game window is open.
